# Сyber Stream Core - это будет публичная часть начала большого проекта, ориентировочное время выполнение проекта от 6-8 месяце на группу из трех разработчиков, а в одного до 12 месяцев.  

**ТЗ прописано в файле REQUIREMENTS.md в данном файле описаны инструкции к запуску, измения и результаты работы**


## Заложил архитектурное и инфраструктурное ядро киберспортивной платформы аналитики (cyber-stream-core). 
Полностью ушёл в сторону реального **highload-домена live-стриминга матчей**. 

**Что сделано в первой публичной части проекта:**

**1. База данных (PostgreSQL + Prisma):** Спроектировал схему под матчи и live-тики игровых событий. Намертво зашил составные B-Tree индексы на связку матчей и таймштампов, чтобы аналитические графики золота и опыта вытаскивались из СУБД за наносекунды без просадки диска. Сделал локальную изоляцию типов Prisma через output = "client"(будет изменятся).

**2. Слой идемпотентности (Redis):** Написал распределенный замок с атомарной командой SET NX. Пакеты защищены жестким TTL (временем жизни) в 15 секунд — это намертво отбивает дубликаты пакетов от лагающего API Steam на самом входе, но при этом автоматически очищает оперативку сервера и защищает систему от переполнения памяти(логика будет добавлена в приватной части).

**3. Бессмертная очередь (RabbitMQ):** Развернул пайплайн на базе продвинутой библиотеки amqp-connection-manager. Теперь если брокер мигнет или упадет во время live-матча — Node.js процесс не крашнется, а обертка сама переподключится и восстановит всю топологию очередей через механизм setup. Настроил отстойник ядовитых сообщений (DLX), чтобы битые пакеты улетали в карантин и не лочили основной live-поток.

**4. Модуль AI (Claude/Anthropic SDK):** Интегрировал Claude API для нелинейной оценки драфтов героев в реальном времени. Написал класс с жестким типом ответа (JSON-парсинг) и предусмотрел режим автоматической демо-эмуляции — если у проверяющего нет платного API-ключа в .env, система подставит пасхальный аналитический вердикт про Коллапса и Магнуса, и код не упадет.

Проект собран как прочный фундамент. Публичную часть зафиксировал, дальше развиваю коммерческую логику в приватном контуре.
